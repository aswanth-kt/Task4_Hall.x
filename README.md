# Authentication System

A full-stack authentication application built as an internship task using React, Node.js, Express, and MongoDB. The application allows users to register, log in, and view their profile after successful authentication.

## Features

* User Registration (Signup)
* User Login Authentication
* MongoDB Database Integration
* React Frontend with Vite
* Tailwind CSS Styling
* Backend API using Express.js
* Frontend-Backend Communication using Axios
* Profile Page Navigation after Login/Signup
* CORS Configuration
* API Testing with Postman

## Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* Axios
* React Router DOM

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* CORS
* Dotenv

## Authentication Flow

1. User registers using Signup form.
2. User data is stored in MongoDB.
3. User logs in with valid credentials.
4. Backend verifies credentials.
5. User is redirected to Profile page.
6. Success and error messages are displayed accordingly.

## Testing

API endpoints were tested using Postman before integrating with the frontend.
